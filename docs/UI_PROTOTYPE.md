# UI 原型设计

## 设计风格

- **设计系统：** Ant Design 5
- **配色方案：**
  - 主色：#1890ff（蓝色）
  - 成功：#52c41a（绿色）
  - 警告：#faad14（橙色）
  - 错误：#ff4d4f（红色）
  - 背景：#f5f5f5（浅灰）
- **字体：** PingFang SC / -apple-system

## 交互设计

### 1. 加载状态
- 使用 Ant Design Spin 组件
- 局部加载：卡片内显示 Loading
- 全局加载：页面居中显示 Loading

### 2. 空状态
- 使用 Ant Design Empty 组件
- 提示引导用户操作（如"添加第一个模型"）

### 3. 错误处理
- 使用 Ant Design message.error() 显示错误
- 表单错误：字段下方显示红色错误提示

### 4. 确认操作
- 删除操作：弹出确认对话框
- 重要操作：二次确认

### 5. 响应式设计
- 移动端：侧边栏折叠，卡片单列显示
- 平板：卡片双列显示
- 桌面：卡片三列显示
